    Simplest RAG Pipeline using LangChain & HuggingFace

    This example walks through how to build a simple Retrieval-Augmented Generation pipeline using LangChain.

    1. Loading Text from .txt File

    from langchain_community.document_loaders import TextLoader

loader = TextLoader("sample.txt")
docs = loader.load()

    2. Loading PDF

    from langchain_community.document_loaders import PyPDFLoader

loader = PyPDFLoader("/content/Survey_Amodal_completion.pdf")
docs = loader.load()

    3. Chunking the Text

    from langchain.text_splitter import RecursiveCharacterTextSplitter

raw_texts = [doc.page_content for doc in docs]

text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=1000,
    chunk_overlap=100
)
texts = text_splitter.create_documents(raw_texts)

    4. Embedding

    from langchain.embeddings import HuggingFaceEmbeddings

embedding = HuggingFaceEmbeddings(model_name="all-MiniLM-L6-v2")

    5. Similarity Search

    from langchain_community.vectorstores import FAISS

db = FAISS.from_documents(texts, embedding)
retriever = db.as_retriever()

    6. Combining Retrieval with Prompt

    retrieved_docs = retriever.get_relevant_documents("What are the related works that have been done on this?")
context = "\n\n".join([doc.page_content for doc in retrieved_docs])

prompt = f"""
You are reading a research paper. The following is extracted content from the abstract and related work sections:

{context}

Please summarize the related work section of the paper, highlighting prior approaches, key methodologies, and how this paper builds upon or differs from them.
"""

response = model.generate_content(prompt)
