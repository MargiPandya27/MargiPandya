    Simplest Implementation of RAG using Langchain and HuggingFace

    With the advent of Natural Language Processing and high-end computational resources, large language models have achieved strong generative capabilities. However, LLMs can become outdated and retraining them frequently is computationally expensive. To generate relevant, updated content efficiently, techniques such as Retrieval-Augmented Generation (RAG), Fine-Tuning, and Prompt Engineering are essential. Each has its own use-case advantages and limitations.

    Comparison Table

    Technique Pros Cons
    Fine-Tuning Good domain understanding Requires frequent retraining, computationally expensive, needs high-quality data
    RAG Context-aware, no need to retrain Can be biased if provided documents lack diversity
    Prompt Engineering Simple, no retraining needed Limited to static training data, less generative depth

    Components of RAG:

    1. Chunking
    2. Embedding
    3. Retrieval
    4. Similarity with the query
    5. Ranking the chunks
    6. Passing to LLM for generation

    Using LangChain for Each Component

    Loading Text from .txt and .pdf Files

    from langchain_community.document_loaders import TextLoader

loader = TextLoader("sample.txt")
docs = loader.load()
  
    from langchain_community.document_loaders import PyPDFLoader

loader = PyPDFLoader("/content/Survey_Amodal_completion.pdf")
docs = loader.load()

    Chunking the Text

    from langchain.text_splitter import RecursiveCharacterTextSplitter

# Extract text content from Document objects
raw_texts = [doc.page_content for doc in docs]

# Split the text into chunks
text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=1000,
    chunk_overlap=100
)
texts = text_splitter.create_documents(raw_texts)

    Embedding

    from langchain.embeddings import HuggingFaceEmbeddings

# Use a lightweight, fast model
embedding = HuggingFaceEmbeddings(model_name="all-MiniLM-L6-v2")

    Similarity Search

    from langchain_community.vectorstores import FAISS

# Create FAISS vectorstore
db = FAISS.from_documents(texts, embedding)
retriever = db.as_retriever()

    Combining Retrieved Content with Prompt

    retrieved_docs = retriever.get_relevant_documents("What are the related works that have been done on this?")
context = "\n\n".join([doc.page_content for doc in retrieved_docs])

prompt = f"""
You are reading a research paper. The following is extracted content from the abstract and related work sections:

{context}

Please summarize the related work section of the paper, highlighting prior approaches, key methodologies, and how this paper builds upon or differs from them.
"""

response = model.generate_content(prompt)
