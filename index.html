    Research Projects

    1. Multiple Foreground Co-segmentation
      Advisor: Dr. Avik Hati
      Multiple foreground common object co-segmentation is a task in computer vision with applications ranging from image understanding to object recognition. The goal is to segment common objects shared among multiple images within a given group. Till now, research in this area has primarily focused on single foreground co-segmentation, where only a single common object is segmented across multiple images. In this project, we work on key approaches such as deep learning-based methods, graph-based algorithms, and optimization techniques to develop an algorithm advised for multiple foreground co-segmentation.
