Variational Autoencoder (VAE) - Likelihood Estimation

A Variational Autoencoder (VAE) is a type of deep generative model that learns to encode data into a latent space and then decode it to generate similar data samples. While originally designed for image data, VAEs can be adapted to work with text, audio, or other modalities.

🔍 Goal

Given a dataset \( X \), we want to learn the parameters \( \theta \) of a generative model such that the likelihood \( p(X; \theta) \) is maximized.

Marginal Likelihood and Latent Variables

To handle complex distributions, VAEs introduce latent variables \( z \) and define a joint distribution \( p(x, z) \). The marginal likelihood of a sample \( x \) is obtained by integrating over all latent variables:

\[ p(x) = \int p(x, z) \, dz \]

This integral is often intractable, especially when \( z \) is high-dimensional. So we estimate it using sampling techniques.

🎲 Sampling Methods

1. Monte Carlo Estimation

We approximate the integral using randomly sampled latent vectors:

\[ p(x) \approx \frac{1}{k} \sum_{i=1}^{k} p(x, z^{(i)}), \quad z^{(i)} \sim \text{Uniform} \]

However, since most sampled \( z \) contribute little to \( p(x, z) \), this method is inefficient.

2. Importance Sampling

We instead sample from a distribution \( q(z) \) that is closer to the posterior:

\[ p(x) = \int \frac{p(x, z)}{q(z)} q(z) \, dz = \mathbb{E}_{z \sim q(z)}\left[\frac{p(x, z)}{q(z)}\right] \]

Approximating with \( k \) samples:

\[ p(x) \approx \frac{1}{k} \sum_{i=1}^{k} \frac{p(x, z^{(i)})}{q(z^{(i)})}, \quad z^{(i)} \sim q(z) \]

📈 Log-Likelihood and ELBO

For i.i.d. dataset \( X = \{x^{(1)}, \dots, x^{(m)}\} \), the log-likelihood is:

\[ \log p(X; \theta) = \sum_{i=1}^{m} \log p(x^{(i)}; \theta) \]

Using Jensen’s Inequality

The log-likelihood for a single point \( x \) can be written using importance sampling:

\[ \log p(x) = \log \mathbb{E}_{z \sim q(z)} \left[ \frac{p(x, z)}{q(z)} \right] \geq \mathbb{E}_{z \sim q(z)} \left[ \log \left( \frac{p(x, z)}{q(z)} \right) \right] \]

This lower bound is called the Evidence Lower Bound (ELBO):

\[ \mathcal{L}(x; \theta, \phi) = \mathbb{E}_{z \sim q_\phi(z|x)} \left[ \log p_\theta(x|z) \right] - D_{KL}(q_\phi(z|x) \| p(z)) \]

Equality Condition

The inequality becomes equality when \( q(z) = p(z|x) \), the true posterior:

\[ \log p(x) = \mathbb{E}_{z \sim p(z|x)} \left[ \log \left( \frac{p(x, z)}{p(z|x)} \right) \right] = \log p(x) \]

🧠 Approximating the Posterior with Neural Networks

Since the true posterior is intractable, we approximate it with a neural network:

\[ q_\phi(z|x) = \mathcal{N}(z; \mu(x), \sigma^2(x)) \]

This is known as the encoder or inference network. It outputs the mean and variance of a Gaussian for each input \( x \).

🧬 Generation Process

  1. Sample \( z \sim p(z) = \mathcal{N}(0, I) \)
  2. Use decoder to generate \( x \sim p_\theta(x|z) \)

The generative model is defined as:

\[ p(x, z) = p(x|z) \cdot p(z) \]

🔧 Final Loss Function

The final VAE loss function (to maximize ELBO) is:

\[ \log p(x) \geq \mathbb{E}_{z \sim q(z|x)} \left[ \log p_\theta(x|z) \right] - D_{KL}(q_\phi(z|x) \| p(z)) \]

📉 KL Divergence Term

Given: \[ q(z|x) = \mathcal{N}(z; \mu, \sigma^2 I), \quad p(z) = \mathcal{N}(z; 0, I) \]

The KL divergence has a closed form:

\[ D_{KL}(q(z|x) \| p(z)) = \frac{1}{2} \sum_{j=1}^{d} \left( \sigma_j^2 + \mu_j^2 - 1 - \log \sigma_j^2 \right) \]
